# student-etl-pipeline
ETL pipeline that extracts, transforms, and loads student data using Python, Pandas, and PostgreSQL.

# Student Data ETL Using Python + PostgreSQL

This project demonstrates a complete ETL (Extract, Transform, Load)
pipeline built using Python, Pandas, and PostgreSQL.

## Features
- Extract student data from CSV files
- Clean and transform data using Pandas
- Load cleaned data into PostgreSQL database
- Perform SQL joins and generate reports
- Export final report using Python
