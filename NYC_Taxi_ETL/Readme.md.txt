NYC Taxi ETL Data Pipeline

This project demonstrates an end-to-end data engineering pipeline built using the NYC Taxi dataset.
  
The pipeline extracts trip data from Parquet files, performs data cleaning and transformation using Python, and loads the processed data into a PostgreSQL database. 
 
Apache Airflow is used to orchestrate and schedule the ETL workflow, while Docker is used to containerize the infrastructure.  

The final dataset is connected to Power BI to build an interactive dashboard for analyzing taxi trips, passenger trends, and trip distances.

This project highlights core data engineering concepts including ETL development, workflow orchestration, containerization, SQL analytics, and data visualization.

Technologies used:

Python, Pandas, Apache Airflow, Docker, PostgreSQL, SQLAlchemy, Power BI.
